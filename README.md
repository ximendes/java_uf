# java-uf

### Pré Requisitos
* Postgres Database
* Criar as bases crud_uf e crud_uf_test 

## Deploy
```
cd app && mvn spring-boot:run
api disponível na portal: 8080
```

## Executar Testes

```
cd app && mvn install
```

## Tecnologias
* Spring Boot 
* FlyWay Database Migration
* Swagger API
* Hibernate 
*JUnit Test
